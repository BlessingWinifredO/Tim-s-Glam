SECTION 5 - EXTERNAL SETUP REQUIRED
==================================================

These cannot be done in code. You must complete each manually.

A. [EXTERNAL] Buy domain on Namecheap
   - Go to namecheap.com, search your desired .com domain name
   - Purchase it (approx $10-15 for year 1)

B. [EXTERNAL] Deploy to Vercel
   - Go to vercel.com, import your GitHub repo
   - Add all environment variables from .env.local to Vercel
   - Connect your Namecheap domain in Vercel project settings
   - Add the DNS records Vercel gives you in Namecheap

C. [EXTERNAL] Setup Google Workspace mailbox
   - Go to workspace.google.com, start Business Starter
   - Verify domain ownership using TXT record in Namecheap
   - Add Google MX records to Namecheap DNS
   - Create your support@yourdomain.com mailbox

D. [EXTERNAL] Setup Resend for email sending
   - Go to resend.com, create a free account
   - Add your domain and verify it (DKIM/SPF records in Namecheap)
   - Create an API key
   - Add to .env.local and Vercel:
     SMTP_HOST=smtp.resend.com
     SMTP_PORT=587
     SMTP_SECURE=false
     SMTP_USER=resend
     SMTP_PASSWORD=your_resend_api_key
     SMTP_FROM=Tim's Glam <support@yourdomain.com>

E. [EXTERNAL] Setup PayPal for payments
   - Go to developer.paypal.com
   - Create a live application and get your live Client ID
   - Add to .env.local and Vercel:
     NEXT_PUBLIC_PAYPAL_CLIENT_ID=your_live_client_id

F. [EXTERNAL] Add all missing env vars to Vercel and .env.local
   NEXT_PUBLIC_APP_URL=https://yourdomain.com
   NEXT_PUBLIC_ADMIN_EMAIL=your_admin_email
   NEXT_PUBLIC_PAYPAL_CLIENT_ID=your_paypal_live_id
   (all the Firebase keys without hardcoded fallbacks)
